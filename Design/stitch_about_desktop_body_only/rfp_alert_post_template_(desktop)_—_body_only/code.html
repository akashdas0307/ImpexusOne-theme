The Dubai Health Authority (DHA) has released a Request for Proposal for the turnkey construction of a new specialized regional health center. This project aims to expand critical care capacity in the Dubai South district by Q4 2025.

Project Scope Summary

  • check_circle Construction of a G+4 main building structure.
  • check_circle Full MEP installation and commissioning.
  • check_circle Landscaping and parking for 200+ vehicles.
  • check_circle Integration with existing district cooling.
