Troubleshooting

Error: "Permission Denied (publickey)" expand_more
This usually happens when your SSH agent isn't forwarding the correct key. Try running ssh-add -K ~/.ssh/id_rsa and restart your terminal.
Webpack fails to compile expand_more
Ensure that you have installed the peer dependencies listed in the package.json manually if you are using npm version 6 or below.
